# نظام إدارة الموارد البشرية

نظام شامل لإدارة الموارد البشرية يتضمن إدارة الطلاب والموظفين والتقارير المفصلة.

## المميزات

- 📊 **تقارير شاملة**: تقارير تفصيلية عن الأشخاص مع مخططات بيانية
- 👥 **إدارة الأشخاص**: إدارة بيانات الطلاب والموظفين
- 📈 **مخطط التقدم الأكاديمي**: رسوم بيانية تُظهر تطور أداء الطلاب
- 💰 **إدارة المالية**: تتبع المدفوعات والمساعدات المالية
- ✅ **نظام الحضور**: تسجيل ومتابعة الحضور اليومي
- 🎓 **المؤهلات الأكاديمية**: إدارة الشهادات والدورات التطويرية

## إنشاء البيانات التجريبية

لتجربة النظام مع بيانات وهمية، قم بتشغيل الأمر التالي:

```bash
./create_sample_data.sh
```

أو يدوياً:

```bash
# إنشاء قاعدة البيانات
python3 manage.py makemigrations
python3 manage.py migrate

# إنشاء البيانات التجريبية الأساسية
python3 manage.py create_sample_data --clear

# إنشاء بيانات التقدم الأكاديمي
python3 manage.py create_academic_progress_data
```

## البيانات التجريبية المُنشأة

عند تشغيل أوامر إنشاء البيانات التجريبية، سيتم إنشاء:

### 📚 **البيانات الأساسية**
- **5 دفعات** دراسية (2020-2024)
- **8 تخصصات** (هندسة حاسوب، تقنية معلومات، إلخ)
- **7 جامعات** يمنية
- **15 مادة دراسية** متنوعة

### 👨‍🎓 **الأشخاص**
- **50 شخص** بأسماء عربية حقيقية
- **33 طالب** و **17 موظف** تقريباً
- بيانات شخصية كاملة (عناوين، هواتف، بطاقات هوية)
- حالات اجتماعية ومالية متنوعة

### 📊 **التقدم الأكاديمي**
- تسجيلات في مواد دراسية عبر السنوات الأربع
- درجات واقعية (80% نجاح، 20% رسوب)
- تطور الأداء عبر الزمن
- بيانات لإنشاء مخططات بيانية

### 💼 **البيانات الإدارية**
- **سجلات حضور** للشهر الماضي
- **مدفوعات مالية** للثلاثة أشهر الماضية
- **15 مخالفة** متنوعة
- **25 دورة تطويرية**
- **30 مؤهل أكاديمي**

## تشغيل النظام

```bash
# تشغيل الخادم المحلي
python3 manage.py runserver

# أو استخدم:
./run_local_server.sh
```

## عرض التقارير

بعد إنشاء البيانات التجريبية:

1. **افتح المتصفح** واذهب إلى: `http://localhost:8000`
2. **اختر أي شخص** من القائمة
3. **شاهد التقرير التفصيلي** مع:
   - المعلومات الشخصية
   - مخطط التقدم الأكاديمي الملون
   - بيانات الحضور والمدفوعات
   - المخالفات والدورات التطويرية

## مخطط التقدم الأكاديمي

المخطط الجديد يعرض:

- **أعمدة ملونة** تمثل متوسط كل فصل دراسي
- **ألوان حسب الأداء**:
  - 🟢 أخضر: ممتاز (85-100)
  - 🔵 أزرق: جيد جداً (75-84)
  - 🟡 أصفر: جيد (65-74)
  - 🟠 برتقالي: مقبول (50-64)
  - 🔴 أحمر: راسب (أقل من 50)
- **إحصائيات تفصيلية** أسفل المخطط

## إعادة إنشاء البيانات

لحذف البيانات الموجودة وإنشاء بيانات جديدة:

```bash
python3 manage.py create_sample_data --clear
python3 manage.py create_academic_progress_data
```
